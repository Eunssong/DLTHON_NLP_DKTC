# Deep Learning 

딥러닝을 활용하여, DKTC 데이터셋을 기반으로 한 4가지 클래스(협박 대화, 갈취 대화, 직장 내 괴롭힘 대화, 기타 괴롭힘 대화) 분류 작업을 진행했습니다.   

사용된 딥러닝 모델은 다음과 같습니다.
- LSTM(Long Short-Term Memory)
- Bidirectional LSTM
- Simple RNN
- GRU
